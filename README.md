# Vicbasertec ASCII Art Animado

Este proyecto es una página web estática que ofrece una herramienta online para convertir imágenes en arte ASCII animado.

## Qué hace

- Convierte imágenes en texto ASCII con estilo visual.
- Aplica efectos animados como `matrix`, `ondas`, `glitch`, `respirar`, `escaneo`, `lluvia`, `neon` y `cyber`.
- Permite ajustar resolución, contraste, brillo y colores.
- Genera salida exportable como HTML/CSS/JS completo.
- Ofrece descarga de resultados en `.txt` y GIF animado.

## Tecnología

- HTML, CSS y JavaScript puro.
- Integración con `gif.js` para creación de GIF animados.
- Diseño orientado a una interfaz moderna y oscura, con soporte de tipografía de Google Fonts.

## Propósito

La idea es proporcionar una experiencia creativa y gratuita para transformar imágenes en composiciones de texto dinámico y animado, ideal para proyectos retro, efectos visuales o solo para divertirse con arte ASCII.

## Ubicación

El archivo principal del proyecto es `index.html` en este directorio.
